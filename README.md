# FitnessExtrem

Web estática lista para subir a GitHub Pages.

## Estructura

- `index.html`
- `styles.css`
- `script.js`
- `img/hero.jpg`
- `img/nutricion.jpg`
- `img/entrenamiento.jpg`
- `img/nutricion-plan.jpg`
- `img/entrenamiento-plan.jpg`
- `img/oferta-conjunta.jpg`

## Publicar en GitHub

1. Sube todos los archivos a un repositorio.
2. Deja `index.html` en la raíz.
3. Activa GitHub Pages desde Settings > Pages.
4. Selecciona la rama principal y la carpeta `/root`.
